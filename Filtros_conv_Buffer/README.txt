#Português:

Implementação de diversos tipos de filtros, incluindo programas para convolução e uso do software winFilter para construção dos filtros digitais. O conteudo de filtros digitais está distribuido da seguinte forma:

- Buffer Circular: é usado para filtragem online
- Convolução; 
- Filtro FIRR com buffer circular;
- Filtro IIR com buffer circular;
- Projeto de filtros com matlab
- Construção de filtros WinFilter:
       * Filtro FIR passa alta;
       * Filtro FIR passa baixa;
       * Filtro IIR passa alta;
       * Filtro IIR passa baixa;

...

#English:

Implementation of several types of filters, including programs for convolution and use of winFilter software to build digital filters. The content of digital filters is distributed as follows:

- Circular Buffer: used for online filtering
- Convolution;
- FIRR filter with circular buffer;
- IIR filter with circular buffer;
- Design of filters with matlab
- Construction of WinFilter filters:
       * High pass FIR filter;
       * Low pass FIR filter;
       * High pass IIR filter;
       * Low pass IIR filter;
