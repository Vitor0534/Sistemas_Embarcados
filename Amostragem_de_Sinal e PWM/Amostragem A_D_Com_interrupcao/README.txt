Código que implementa uma amostragem de sinal com uso de interrupção e configuração de registradores. 

o programa está configurado para lançar uma interrupção com timer de 0,1 segundos. Do mesmo modo, o programa contem opções para se configurar o prescaler, este, controla o clock do conversor A/D e serve para configurar os pulsos de clock necessários para conversão A/D.
